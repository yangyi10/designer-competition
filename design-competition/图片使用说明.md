# 图片资源使用说明

## 图片插入方式

### 1. 本地图片（推荐）

1. 将图片文件放到 `images` 文件夹中
2. 在 `script.js` 中修改图片路径：

```javascript
{
    id: 1,
    title: '作品标题',
    image: 'images/work1.jpg'  // 相对路径
}
```

**支持的图片格式：** JPG, PNG, GIF, WebP, SVG

### 2. 网络图片URL

直接使用网络图片的完整URL：

```javascript
{
    id: 1,
    title: '作品标题',
    image: 'https://example.com/your-image.jpg'
}
```

### 3. API生成图片

使用现有的API方式，修改prompt参数：

```javascript
{
    id: 1,
    title: '作品标题',
    image: 'https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=你的描述&image_size=landscape_16_9'
}
```

### 4. Base64编码图片

将图片转换为Base64字符串：

```javascript
{
    id: 1,
    title: '作品标题',
    image: 'data:image/jpeg;base64,/9j/4AAQSkZJRgABAQAAAQABAAD...'
}
```

## 图片尺寸建议

- **缩略图：** 280x200px（首页卡片显示）
- **详情图：** 800x450px（详情弹窗显示）
- **建议比例：** 16:9 或 4:3

## 批量替换图片

如果要替换所有图片，可以按以下步骤：

1. 准备24张图片，命名为 `work1.jpg` 到 `work24.jpg`
2. 将所有图片放入 `images` 文件夹
3. 修改 `script.js` 中的所有图片路径：

```javascript
// 使用查找替换功能
// 查找: image: 'https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt=.*&image_size=landscape_16_9'
// 替换: image: 'images/work{id}.jpg'
```

## 注意事项

1. 本地图片路径区分大小写
2. 图片文件名不要包含中文字符
3. 建议使用JPG或PNG格式以获得最佳兼容性
4. 图片文件大小建议控制在500KB以内
